Selamat datang di repositori proyek web pertama saya! Ini adalah platform *landing page* interaktif yang dirancang khusus untuk memamerkan jasa dan layanan freelance secara profesional. Proyek ini merupakan tonggak awal perjalanan saya dalam dunia pengembangan web.

---

## 📌 Tentang Proyek
Website ini dibangun sebagai solusi digital bagi para penyedia jasa (Freelancer) untuk membangun *personal branding*. Fokus utama proyek ini adalah pada kesederhanaan navigasi dan penyampaian informasi layanan yang efektif kepada klien potensial.

> **"Setiap ahli dulunya adalah seorang pemula."**
> Proyek ini adalah bukti dedikasi saya dalam mempelajari fundamental web desain dan struktur antarmuka yang ramah pengguna.

---

## ✨ Fitur Utama

### 🖥️ Frontend Experience
* **Landing Page Dinamis (`home.html`):** Halaman utama dengan *tagline* profesional dan sambutan hangat untuk pengunjung.
* **Service Showcase (`services.html`):** Daftar layanan yang ditawarkan secara detail, mulai dari jasa pemrograman hingga desain kreatif.
* **Interactive Portfolio (`portofolio.html`):** Galeri visual yang menampilkan hasil karya menggunakan aset gambar dan GIF interaktif.
* **Detailed Profile (`about.html`):** Informasi mendalam mengenai latar belakang, keahlian, dan visi pengembang.
* **Contact Gateway (`contact.html`):** Halaman khusus yang memudahkan klien untuk terhubung langsung dengan penyedia jasa.

### 🎨 Desain & Aset
* **Responsive Layout:** Menggunakan teknik CSS untuk memastikan tampilan tetap rapi di berbagai ukuran layar.
* **Rich Media:** Penggunaan aset visual (GIF, JPEG, Icon) yang tersusun rapi di folder `images/` untuk meningkatkan estetika *user interface*.

---

## 🛠️ Stack Teknologi
Proyek ini dibangun menggunakan teknologi *client-side* murni untuk memastikan kecepatan akses:
* **HTML5:** Sebagai kerangka utama struktur konten.
* **CSS3:** Untuk styling, tata letak modern, dan animasi transisi.
* **JavaScript:** Digunakan untuk meningkatkan interaktivitas elemen web.
